I'm a Computer Science and Finance student at the University of Calgary, interested in building technology that solves practical, high-impact problems.

My work spans full-stack applications, applied machine learning, financial data systems, and business automation. I especially enjoy taking complex or inefficient processes and turning them into software that is clear, reliable, and useful.

Beyond coursework, I have built software for a real retail business, developed financial machine-learning pipelines, explored global economic data, and competed in fast-paced software hackathons.
